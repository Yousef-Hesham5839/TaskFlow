# Flutter Task Manager App 🚀

A clean and modern Task Management app built with Flutter.

## ✨ Features

- Add, delete, and manage tasks
- Categorized tasks (To Do / Done)
- Dark & Light mode support
- Multi-screen navigation
- Local storage using SharedPreferences
- Responsive and clean UI design

## 🛠️ Tech Stack

- Flutter
- Dart
- SharedPreferences
- Material Design

## 🎯 Purpose

This project was built as a portfolio project to demonstrate real-world Flutter development skills including UI design, state management, and local data persistence.
